# 🤖 chaobot 开发计划

## 项目概述

chaobot 是一个轻量级个人 AI 助手，灵感来源于 nanobot，但采用更现代化的架构设计，解决了原项目的一些潜在问题。

## 📝 注意事项

1. **保持简洁**: 参考 nanobot 的轻量级理念，避免过度工程化
2. **类型安全**: 使用 Python 3.10+ 的类型注解
3. **异步优先**: 所有 I/O 操作使用 asyncio
4. **配置驱动**: 通过配置文件控制功能，减少硬编码
5. **向后兼容**: 版本升级时保持配置兼容
