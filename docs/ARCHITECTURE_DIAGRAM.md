    ▼

## 七、扩展点说明

```
┌─────────────────────────────────────────────────────────────────┐
│                         扩展点指南                               │
└─────────────────────────────────────────────────────────────────┘

1. 添加新工具
   └── 继承 BaseTool → 注册到 ToolRegistry
   文件: chaobot/agent/tools/<your_tool>.py

2. 添加新渠道
   └── 继承 BaseChannel → 注册到 ChannelManager
   文件: chaobot/channels/<your_channel>.py

3. 添加新 Provider
   └── 配置添加到 ProviderRegistry.PROVIDERS
   文件: chaobot/providers/registry.py

4. 添加新技能
   └── 创建 SKILL.md 文件
   文件: chaobot/skills/<skill_name>/SKILL.md

5. 添加子代理执行器
   └── 实现 executor 函数 → 注册到 TASK_EXECUTORS
   文件: chaobot/agent/subagent.py
```
